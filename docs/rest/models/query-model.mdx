## Description

This API provides a REST endpoint for making predictions. 
<Note>
By default, the open source version doesn't provide Authentication methods as MindsDB Cloud. 
On MindsDB cloud you need to set the `cookie` field in the header of request e.g  `{session=273trgsehgrui3i2riurwehe}`. 
</Note>

<RequestExample>

```shell Shell
curl --request POST \
     --url https://cloud.mindsdb.com/api/projects/{projectName}/models/{modelName}/predict \
     --header 'Content-Type: application/json' \
     --cookie '{session=273trgsehgrui3i2riurwehe}'\
     --data '
{
   'data': [
       {'sqft': '1000'},
       {'sqft': '500'},
    ],
   'params': {

   }
}
```


```python Python
import requests
url = 'https://cloud.mindsdb.com/projects/{projectName}/models/{modelName}/predict'
cookies = {'session': '273trgsehgrui3i2riurwehe'}
data = { 'data': [{'sqft': '1000'}, {'sqft': '500'}],'params': {}}
resp = requests.post(url, data=data, cookies=cookies)
```

</RequestExample>
